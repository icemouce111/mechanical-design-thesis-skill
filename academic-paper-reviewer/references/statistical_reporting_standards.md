# Statistical Reporting Standards — 統計報告標準與 APA 7.0 格式速查

本文件定義量化研究統計報告的完整審查標準。`methodology_reviewer_agent` 在 Step 4a（Statistical Reporting Adequacy）中使用此文件作為主要參照依據。

---

## 1. Universal Statistical Reporting Checklist（通用統計報告 Checklist）

所有量化研究論文**必須**報告以下項目。審查時逐項檢核：

### 1.1 描述性統計（Descriptive Statistics）

| 項目 | 標準 | 常見缺失 |
|------|------|---------|
| 平均數（*M*） | 所有連續變項均需報告 | 只報告整體，未報告分組 |
| 標準差（*SD*） | 與平均數成對出現 | 誤用標準誤（*SE*）取代 |
| 樣本數（*N* / *n*） | 總樣本 + 分組樣本均需報告 | 分析過程中樣本流失未說明 |
| 全距 / 範圍（Range） | 報告 Min–Max 或四分位距 | 完全缺失，無法判斷分佈特性 |
| 類別變項分佈 | 報告頻率（*f*）和百分比（%） | 只報告百分比，缺少原始頻率 |

### 1.2 效果量（Effect Size）

| 項目 | 標準 | 常見缺失 |
|------|------|---------|
| 報告義務 | **所有統計檢定必須伴隨效果量** — APA 7.0 強制要求 | 僅報告 *p*-value，未報告效果量 |
| 選擇適當指標 | 依分析方法選擇對應的效果量指標（見 §2） | 使用不當的效果量指標 |
| 解讀 | 需提供 Cohen 的慣例基準或領域特定基準 | 報告數值但不解讀大小意義 |

**常用效果量指標速查：**

| 分析方法 | 效果量指標 | 小/中/大（Cohen 慣例） |
|---------|----------|---------------------|
| *t*-test | Cohen's *d* | 0.2 / 0.5 / 0.8 |
| ANOVA | *η*² (eta-squared) | .01 / .06 / .14 |
| ANOVA（偏） | *η*²_p (partial) | .01 / .06 / .14 |
| 相關 | *r* | .10 / .30 / .50 |
| 迴歸 | *R*², *f*² | *f*²: .02 / .15 / .35 |
| 卡方 | Cramér's *V*, *φ* | *V*: .10 / .30 / .50（*df*=1） |
| Odds Ratio | OR | 1.5 / 2.5 / 4.3（Rosenthal） |

### 1.3 信賴區間（Confidence Intervals）

| 項目 | 標準 | 常見缺失 |
|------|------|---------|
| CI 報告 | 所有效果量和關鍵估計值**應**報告 95% CI | 完全未報告 CI |
| 格式 | 95% CI [下界, 上界] | 格式不一致或使用括號而非方括號 |
| 解讀 | 描述 CI 的實質意義，不只是統計意義 | 只看 CI 是否包含零，未解讀寬度 |

### 1.4 統計顯著性（Statistical Significance）

| 項目 | 標準 | 常見缺失 |
|------|------|---------|
| *p*-value 格式 | 報告精確 *p* 值（如 *p* = .032） | 僅報告 *p* < .05 或 *p* > .05 |
| *p* < .001 | 當 *p* 非常小時可報告 *p* < .001 | 報告 *p* = .000（統計軟體原始輸出） |
| α 水準 | 事前宣告 α 水準 | 未說明使用 α = .05 或其他值 |
| 多重比較 | 使用 Bonferroni、Holm、FDR 校正 | 多次比較未做任何校正 |
| 不顯著結果 | 完整報告，不可隱藏 | 選擇性只報告顯著結果 |

### 1.5 統計考驗力（Statistical Power）

| 項目 | 標準 | 常見缺失 |
|------|------|---------|
| 事前 power analysis | 說明目標 power（通常 ≥ .80）、預設效果量、α、所需樣本量 | 完全未做 power analysis |
| 效果量來源 | 基於先前研究、pilot study、或理論預期 | 使用 Cohen 慣例但未說明理由 |
| 工具 | 使用 G*Power、pwr package 等 | 未說明使用何種工具 |
| 事後 power | 對不顯著結果報告 observed power | 不顯著時未討論 Type II error 風險 |
| 敏感度分析 | 報告在給定 *N* 下可偵測的最小效果量 | 未做敏感度分析 |

### 1.6 缺失值處理（Missing Data）

| 項目 | 標準 | 常見缺失 |
|------|------|---------|
| 缺失量報告 | 報告每個變項的缺失量和比例 | 未報告缺失狀況 |
| 缺失機制 | 討論 MCAR / MAR / MNAR | 假設 MCAR 但未檢驗 |
| 處理方法 | 說明使用的方法：listwise deletion / pairwise deletion / MI / FIML | 未說明或僅使用 listwise deletion |
| 敏感度分析 | 比較不同缺失處理方法的結果穩健性 | 只用一種方法，未檢驗敏感度 |

### 1.7 假設檢驗前提（Assumption Testing）

| 假設 | 適用分析 | 檢驗方法 | 常見缺失 |
|------|---------|---------|---------|
| 常態性（Normality） | *t*-test, ANOVA, 迴歸 | Shapiro-Wilk / K-S / Q-Q plot / skewness & kurtosis | 完全未檢驗或僅以 CLT 搪塞 |
| 變異數同質性（Homogeneity） | 獨立 *t*-test, ANOVA | Levene's test | 未報告或違反時未使用替代方法 |
| 線性（Linearity） | 迴歸, 相關 | 殘差圖 / scatter plot | 未檢驗即假設線性關係 |
| 獨立性（Independence） | 多數參數檢定 | Durbin-Watson / 研究設計說明 | 巢套資料（nested data）未處理 |
| 共線性（Multicollinearity） | 多元迴歸 | VIF, tolerance, 相關矩陣 | VIF 未報告或報告但未處理 |
| 殘差常態 / 等分散（Residuals） | 迴歸 | 殘差圖, Breusch-Pagan | 模型配適後未檢查殘差 |

## 2. Method-Specific Checklists（按分析方法分類的 Checklist）

### 2.1 *t*-test（獨立樣本 / 配對樣本）

| 檢查項目 | 說明 |
|---------|------|
| 報告 *t* 統計量 | *t*(df) = X.XX, *p* = .XXX |
| 獨立 vs 配對 | 是否正確選擇？配對設計需報告配對邏輯 |
| 效果量 | Cohen's *d*（獨立）或 *d*_z（配對） |
| 前提檢驗 | 常態性（小樣本時重要）、變異數同質性（獨立 *t*-test） |
| Welch's *t*-test | 變異數不同質時是否使用 Welch 校正？ |
| 方向性 | 單尾 vs 雙尾是否有事前理論依據？ |

### 2.2 ANOVA（單因子 / 多因子 / 重複量測）

| 檢查項目 | 說明 |
|---------|------|
| 報告 *F* 統計量 | *F*(df₁, df₂) = X.XX, *p* = .XXX |
| 效果量 | *η*², *η*²_p, 或 *ω*² |
| 事後比較 | 主效果顯著時是否做 post-hoc（Tukey / Bonferroni / Games-Howell）？ |
| 交互作用 | 多因子設計是否解讀交互作用？是否做 simple effects？ |
| 球形假設 | 重複量測設計是否報告 Mauchly's test + Greenhouse-Geisser / Huynh-Feldt 校正？ |
| 前提檢驗 | 常態性、變異數同質性（Levene's）、組間觀察值獨立 |
| 不等組樣本 | 組間人數差異大時是否使用 Type III SS？ |

### 2.3 迴歸分析（線性 / Logistic）

#### 線性迴歸（Linear Regression）

| 檢查項目 | 說明 |
|---------|------|
| 模型摘要 | *R*², adjusted *R*², *F* test for model |
| 係數表 | *B*, *SE*, *β*, *t*, *p*, 95% CI for *B* |
| 共線性 | VIF（< 5 或 < 10 依領域慣例）, tolerance |
| 殘差診斷 | 常態性、等分散性、線性、離群值（Cook's *D*） |
| 變項選取 | 進入法（enter）、逐步法（stepwise）的理由 |
| 效果量 | *R*², *f*², Cohen's *f*² |

#### Logistic 迴歸（Logistic Regression）

| 檢查項目 | 說明 |
|---------|------|
| 模型配適 | Hosmer-Lemeshow / χ² / -2LL / Nagelkerke *R*² |
| 係數報告 | *B*, *SE*, Wald, OR, 95% CI for OR |
| 分類準確度 | Classification table, sensitivity, specificity, AUC/ROC |
| 假設 | 觀察值獨立、線性 logit（連續預測變項與 logit 的線性關係） |
| 樣本量 | 每個預測變項至少 10–20 個事件（EPV 規則） |

### 2.4 結構方程模型（SEM）

| 檢查項目 | 標準 |
|---------|------|
| 樣本量 | 通常 ≥ 200；或 5–10 倍於待估計參數 |
| 模型配適指標 | **必須同時報告多個指標**（至少 4 個） |
| CFI / TLI | ≥ .95（良好）；≥ .90（可接受） |
| RMSEA | ≤ .06（良好）；≤ .08（可接受）；需報告 90% CI |
| SRMR | ≤ .08 |
| χ²/df | ≤ 3（部分學者建議 ≤ 2） |
| 因素負荷量 | 標準化 ≥ .50（理想 ≥ .70） |
| 測量模型 | CFA 先於 SEM（two-step approach）— Anderson & Gerbing (1988) |
| 信效度 | CR ≥ .70, AVE ≥ .50, 區辨效度（Fornell-Larcker / HTMT） |
| 修正指標 | 使用 modification indices 時需有理論支撐 |
| 常態性 | 多變量常態性（Mardia's coefficient）；違反時使用 robust ML 或 bootstrapping |

### 2.5 階層線性模型（HLM / MLM）

| 檢查項目 | 標準 |
|---------|------|
| 巢套結構 | 清楚說明各層級（如：學生→班級→學校） |
| ICC | 報告 Intraclass Correlation Coefficient；ICC > .05 支持使用 MLM |
| 隨機效果 | 報告隨機截距和（如適用）隨機斜率的變異數 |
| 固定效果 | 報告係數、*SE*、*t* / *z*、*p*、CI |
| 組間樣本量 | Level-2 單位數（通常建議 ≥ 30） |
| 置中（Centering） | 說明使用 grand-mean centering 或 group-mean centering 及理由 |
| 模型比較 | 使用 deviance (-2LL)、AIC、BIC 比較巢套模型 |
| 效果量 | Pseudo *R*²（如 Snijders & Bosker's *R*²） |

### 2.6 卡方檢定（Chi-Square Test）

| 檢查項目 | 說明 |
|---------|------|
| 報告格式 | χ²(df, *N* = XX) = X.XX, *p* = .XXX |
| 效果量 | Cramér's *V*（多於 2×2）或 *φ*（2×2） |
| 期望次數 | 所有格的期望次數 ≥ 5；若有格 < 5，使用 Fisher's exact test |
| 獨立性 | 觀察值是否真正獨立？（重複量測不適用普通卡方） |
| 殘差分析 | 顯著時需檢查 standardized residuals 以確定哪些格貢獻顯著 |

### 2.7 非參數檢定（Non-parametric Tests）

| 檢查項目 | 說明 |
|---------|------|
| 使用理由 | 清楚說明為何不使用參數檢定（如常態性違反、順序尺度） |
| 方法選擇 | Mann-Whitney *U* / Wilcoxon / Kruskal-Wallis / Friedman — 是否正確對應 |
| 效果量 | *r* = *Z* / √*N*（Mann-Whitney）；*W*（Kendall's） |
| 報告格式 | 報告檢定統計量、*p*-value、效果量 |
| 事後比較 | Kruskal-Wallis 顯著後需做 pairwise 比較 + 校正 |

## 3. APA 7th Edition 統計格式速查表

### 3.1 數字格式

| 規則 | 正確 | 錯誤 |
|------|------|------|
| *p*-value 無前導零 | *p* = .032 | *p* = 0.032 |
| 能超過 1.0 的統計量有前導零 | *M* = 0.75 | *M* = .75 |
| 不能超過 1.0 的統計量無前導零 | *r* = .45 | *r* = 0.45 |
| 小數位數一般為 2 位 | *M* = 3.45 | *M* = 3.4 |
| *p*-value 小數 2-3 位 | *p* = .03 或 *p* = .032 | *p* = .0321 |
| 百分比 0-1 位小數 | 45.2% | 45.2381% |

**不能超過 1.0 的統計量**（無前導零）：相關係數（*r*, *R*）、比率（*p*-value）、Cramér's *V*、*φ*、*η*²、*R*²、*β*（標準化迴歸係數）

**可超過 1.0 的統計量**（有前導零）：*M*, *SD*, *B*（非標準化迴歸係數）, Cohen's *d*, *t*, *F*, χ²

### 3.2 統計符號斜體規則

| 斜體（*italic*） | 非斜體 |
|-----------------|--------|
| *M*, *SD*, *SE* | df |
| *N*（總樣本）, *n*（子樣本） | SS, MS |
| *t*, *F*, *p*, *r*, *R*, *z* | OR, CI, VIF |
| *d*, *f*², *η*², *ω*² | AIC, BIC, CFI, TLI |
| *B*, *β* | RMSEA, SRMR |
| *χ*² | ICC |
| *U*, *W*（非參數統計量） | ANOVA, SEM, HLM |

### 3.3 統計結果報告格式範例

| 分析方法 | APA 格式範例 |
|---------|-------------|
| 獨立樣本 *t*-test | *t*(58) = 2.45, *p* = .017, *d* = 0.63, 95% CI [0.12, 1.14] |
| 配對樣本 *t*-test | *t*(29) = -3.12, *p* = .004, *d*_z = 0.57 |
| 單因子 ANOVA | *F*(2, 87) = 4.56, *p* = .013, *η*²_p = .09 |
| 線性迴歸 | *B* = 0.34, *SE* = 0.12, *β* = .28, *t*(95) = 2.83, *p* = .006, 95% CI [0.10, 0.58] |
| Logistic 迴歸 | *B* = 1.24, *SE* = 0.45, Wald = 7.59, *p* = .006, OR = 3.46, 95% CI [1.43, 8.37] |
| 卡方 | χ²(2, *N* = 150) = 8.34, *p* = .015, *V* = .24 |
| Mann-Whitney | *U* = 245.00, *z* = -2.31, *p* = .021, *r* = .29 |
| SEM 配適 | χ²(52) = 78.34, *p* = .011, CFI = .97, TLI = .96, RMSEA = .045 [.018, .068], SRMR = .038 |
| HLM 固定效果 | *γ*₁₀ = 0.45, *SE* = 0.15, *t*(28) = 3.00, *p* = .006 |

### 3.4 表格格式標準

| 規則 | 說明 |
|------|------|
| 三線表 | APA 表格只有三條水平線（表頭上方、表頭下方、表底），無垂直線 |
| 表格編號 | Table 1, Table 2...（粗體），標題在編號下一行（斜體） |
| 註解層級 | 一般註（Note.）→ 特定註（ᵃ, ᵇ）→ 顯著性（*p* < .05, **p* < .01） |
| 星號 | \**p* < .05. \*\**p* < .01. \*\*\**p* < .001. |
| 對齊 | 數字右對齊，小數點對齊 |

## 4. Statistical Red Flags（統計「紅旗」清單）

審查時，以下模式應引起警覺，需進一步調查或要求作者釐清：

### 4.1 P-hacking 指標

| 紅旗 | 描述 | 嚴重度 |
|------|------|--------|
| 大量 *p* ≈ .05 | 多個結果的 *p* 集中在 .04–.05 區間 | HIGH |
| 選擇性報告 | 只報告顯著結果，不顯著的消失了 | HIGH |
| 分析策略模糊 | 未事先說明分析策略，事後看起來像是探索 | MEDIUM |
| 意外子群體 | 事後拆解子群體以找到顯著結果 | MEDIUM |
| 彈性樣本量 | 未事前定義停止規則（sequential testing without correction） | HIGH |
| 「排除離群值」 | 大量排除離群值但標準不明確 | MEDIUM |

### 4.2 HARKing（Hypothesizing After Results are Known）

| 紅旗 | 描述 | 嚴重度 |
|------|------|--------|
| 假設與結果完美吻合 | 所有假設都被支持，無一例外 | MEDIUM |
| 探索性分析包裝為驗證性 | 文獻回顧明顯是事後建構的 | HIGH |
| 假設方向性變更 | 原本預期正向但結果是負向，卻宣稱「符合預期」 | HIGH |
| 缺少預註冊 | 未提供 OSF / AsPredicted 等預註冊連結（非強制但建議） | LOW |

### 4.3 效果量與信賴區間缺失

| 紅旗 | 描述 | 嚴重度 |
|------|------|--------|
| 完全不報告效果量 | 只看 *p*-value 做結論 | HIGH |
| CI 完全缺失 | 無法判斷估計精確度 | MEDIUM |
| CI 極寬 | CI 包含從小到大的效果量，估計不精確 | MEDIUM |
| 效果量不一致 | 報告的效果量與原始數據推算不符 | HIGH |

### 4.4 樣本量問題

| 紅旗 | 描述 | 嚴重度 |
|------|------|--------|
| 無 power analysis | 樣本量缺乏事前計算依據 | MEDIUM |
| 樣本過小 | 迴歸分析中 *N* < 10 × 預測變項數 | HIGH |
| 樣本流失未說明 | 起始 *N* 和最終 *N* 差距大但未說明原因 | MEDIUM |
| SEM 小樣本 | *N* < 200 且未使用小樣本校正 | MEDIUM |
| HLM Level-2 不足 | Level-2 單位 < 30 | MEDIUM |

### 4.5 多重比較未校正

| 紅旗 | 描述 | 嚴重度 |
|------|------|--------|
| 多次 *t*-test 替代 ANOVA | 3 組以上比較用多次 *t*-test | HIGH |
| ANOVA 後無 post-hoc | 主效果顯著但未做事後比較就宣稱組間差異 | MEDIUM |
| 多個依變項無校正 | 同一資料集多個依變項分別檢定，未做 Bonferroni 或 FDR | MEDIUM |
| 多重模型比較 | 嘗試多個模型但只報告「最好的那一個」 | HIGH |

### 4.6 假設檢驗前提違反

| 紅旗 | 描述 | 嚴重度 |
|------|------|--------|
| 完全未報告前提檢驗 | 跳過常態性/同質性/線性等檢驗 | MEDIUM |
| 前提違反但不處理 | 報告了違反但照用原分析 | HIGH |
| 以 CLT 搪塞 | 「因為 *N* > 30 所以可忽略常態性」但未實際檢驗 | LOW |
| VIF 過高 | VIF > 10 但未做任何處理 | HIGH |

### 4.7 其他紅旗

| 紅旗 | 描述 | 嚴重度 |
|------|------|--------|
| *p* = .000 | 統計軟體輸出原始值，應為 *p* < .001 | LOW |
| df 與 *N* 不符 | 自由度推算出的 *N* 與報告 *N* 不一致 | HIGH |
| 表格內數字不一致 | 文本敘述與表格數值矛盾 | HIGH |
| 統計軟體未說明 | 未報告使用 SPSS / R / Stata / Mplus 及版本 | LOW |
| 因果語言 | 非實驗設計（相關/調查）使用因果推論語言 | MEDIUM |

## 5. 高教研究常見統計方法推薦

高等教育研究（Higher Education Research）領域的論文經常涉及以下議題與對應的分析方法。審查時可參考此表判斷方法選擇是否適當。

### 5.1 依研究問題類型推薦方法

| 研究問題類型 | 推薦方法 | 說明 |
|------------|---------|------|
| 兩組比較（如實驗 vs 控制） | 獨立樣本 *t*-test / Mann-Whitney | 視資料常態性決定 |
| 多組比較（如不同學校類型） | ANOVA / Kruskal-Wallis | 3+ 組的平均數比較 |
| 前後測比較 | 配對 *t*-test / Wilcoxon | 同一群體的前後變化 |
| 預測分析（連續依變項） | 多元迴歸 | 多個預測變項對連續結果的影響 |
| 預測分析（二元依變項） | Logistic 迴歸 | 如：畢業/輟學、通過/未通過 |
| 巢套資料（學生→學校） | HLM / MLM | 高教資料天然有巢套結構 |
| 潛在構念與路徑分析 | SEM / CFA | 測量不可直接觀察的構念（如教學品質） |
| 問卷信效度 | EFA → CFA | 量表開發或驗證 |
| 類別變項關聯 | 卡方 / Fisher's exact | 交叉表分析 |
| 縱貫資料 | 成長曲線模型 / 潛在成長模型 | 追蹤學生多年的變化軌跡 |
| 大規模資料集 | 加權分析 / 抽樣設計校正 | 使用全國調查資料時需考慮抽樣設計 |

### 5.2 高教研究特殊注意事項

| 注意事項 | 說明 |
|---------|------|
| **巢套結構** | 高教資料幾乎都有巢套（學生→科系→學校）；忽略會低估標準誤、膨脹 Type I error |
| **抽樣設計** | 使用全國性資料庫（如 MOE 統計、高教公共化資料）時，必須考慮抽樣權重和 clustering |
| **選擇偏誤** | 學生自選科系/學校，非隨機分派；應考慮 propensity score matching 或 Heckman correction |
| **天花板效應** | 滿意度調查常出現極端偏態；需檢查並考慮 Tobit model 或非參數方法 |
| **小母體** | 台灣大學數量有限（~150 所）；全數調查時不適用推論統計（是普查不是抽樣） |
| **時間序列** | 分析多年招生/註冊趨勢時需考慮自相關（autocorrelation） |
| **多重角色** | 同一教師填寫多份問卷（如教學評鑑）→ 觀察值不獨立 |

## 6. 統計報告完整性評分標準

`methodology_reviewer_agent` 使用以下標準評估統計報告完整性：

### 評分維度與權重

| 維度 | 權重 | 滿分項目 |
|------|------|---------|
| A. 描述性統計完整性 | 15% | M, SD, N, Range 齊全 |
| B. 效果量報告 | 20% | 所有檢定均伴隨效果量 |
| C. 信賴區間報告 | 15% | 關鍵估計值附 CI |
| D. 前提檢驗報告 | 15% | 所有統計假設均有檢驗 |
| E. 統計考驗力 | 10% | 事前 power analysis 完整 |
| F. 缺失值處理 | 10% | 報告缺失量 + 處理方法 |
| G. APA 格式正確性 | 10% | 符號、小數、表格合規 |
| H. 無紅旗指標 | 5% | 未發現 §4 列出的紅旗 |

### 評分等級

| 等級 | 分數 | 描述 |
|------|------|------|
| Exemplary | 90–100 | 統計報告堪為典範，所有項目齊全且格式正確 |
| Adequate | 70–89 | 主要項目齊全，少數遺漏但不影響結論可信度 |
| Needs Improvement | 50–69 | 有顯著遺漏（如缺效果量或前提檢驗），需補充 |
| Inadequate | 30–49 | 多項缺失，統計報告不足以支撐結論 |
| Unacceptable | 0–29 | 統計報告嚴重不足，需要大幅重寫 |

## 7. 快速參考：審查順序建議

Methodology reviewer 在審查統計報告時，建議依以下順序進行：

```
Step 1: 確認研究問題→分析方法的對應是否合理（§5）
Step 2: 檢查假設檢驗前提是否報告（§1.7）
Step 3: 逐項檢查通用 checklist（§1.1–1.6）
Step 4: 查對應方法的專屬 checklist（§2）
Step 5: 掃描紅旗清單（§4）
Step 6: 驗證 APA 格式（§3）
Step 7: 產出完整性評分（§6）
```
