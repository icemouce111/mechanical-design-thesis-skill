# Funding Statement Guide — 資金來源聲明指南

Used by `intake_agent`, `formatter_agent`, and `draft_writer_agent`.

## 概述

Funding statement（資金來源聲明）是學術論文中必備的聲明之一，無論研究是否接受資助，均需明確說明。多數期刊要求在投稿時提供 funding 資訊，並在稿件中包含正式聲明。本指南涵蓋台灣與國際主要資助機構的格式規範、聲明範本，以及與 COI 聲明的區別。

---

## 台灣資助機構格式

### 1. NSTC（國家科學及技術委員會）

**前身**：MOST（科技部，Ministry of Science and Technology），2022 年 7 月改制為 NSTC。

**計畫編號格式**：`NSTC [年度]-[學門代碼]-[類別]-[流水號]-[分項]`

| 欄位 | 說明 | 範例 |
|------|------|------|
| 年度 | 民國年 3 碼 | 113 |
| 學門代碼 | 學門分類代碼 | 2410（教育學門） |
| 類別 | 計畫類型 | H（專題研究） |
| 流水號 | 申請序號 | 001 |
| 分項 | 子計畫（若有） | MY2（多年期第 2 年） |

**常見學門代碼**（高教研究相關）：

| 代碼 | 學門 |
|------|------|
| 2410 | 教育學 |
| 2420 | 心理學 |
| 2422 | 管理學 |
| 2418 | 社會學 |

**英文格式範例**：
```
This work was supported by the National Science and Technology Council, Taiwan
(Grant No. NSTC 113-2410-H-003-001).
```

**中文格式範例**：
```
本研究承國家科學及技術委員會專題研究計畫補助（計畫編號：NSTC 113-2410-H-003-001），特此致謝。
```

**注意事項**：
- 2022 年 7 月以前的計畫使用 MOST 編號（如 MOST 110-2410-H-003-001），引用時保留原編號
- 多年期計畫需標註年度（如 MY2 表示第 2 年）
- 同一篇論文涉及多個計畫時，依主要性排序列出

### 2. MOE（教育部）

**常見計畫類型**：

| 計畫 | 英文名稱 | 編號格式 |
|------|---------|---------|
| 高教深耕計畫 | Higher Education Sprout Project | MOE-[年度]-S-[校碼] |
| 教學實踐研究計畫 | Teaching Practice Research Program | PBM[年度][序號] |
| 大學社會責任實踐計畫（USR） | University Social Responsibility Project | MOE-USR-[年度]-[序號] |
| 玉山學者計畫 | Yushan Scholar Program | -- |
| 學海系列計畫 | Study Abroad Programs | -- |

**英文格式範例**：
```
This work was supported by the Ministry of Education, Taiwan, through the
Higher Education Sprout Project (Grant No. MOE-113-S-0023).
```

**中文格式範例**：
```
本研究承教育部高等教育深耕計畫補助（計畫編號：MOE-113-S-0023），特此致謝。
```

### 3. 校內計畫

**常見類型**：

| 計畫類型 | 說明 |
|---------|------|
| 校內研究發展基金 | 各校研發處管理的校內補助 |
| 教學研究計畫 | 教務處或教學發展中心補助 |
| 新進教師研究補助 | 針對新進教師的啟動經費 |
| 校內跨領域研究計畫 | 鼓勵跨院系合作 |

**英文格式範例**：
```
This work was supported by a research grant from [University Name]
(Grant No. [校內編號]).
```

**中文格式範例**：
```
本研究承[校名]校內研究計畫補助（計畫編號：[校內編號]），特此致謝。
```

do not necessarily reflect the views of the National Science Foundation.

## 聲明範本

### 有資助（單一來源）

**英文**：
```
Funding: This work was supported by the [Funder Full Name] [Grant Type]
(Grant No. [Number]).
```

**中文**：
```
經費來源：本研究承[資助機構全稱][計畫類型]補助（計畫編號：[編號]），特此致謝。
```

### 有資助（多重來源）

**英文**：
```
Funding: This work was supported by the [Funder 1] (Grant No. [Number 1]);
the [Funder 2] (Grant No. [Number 2]); and the [Funder 3] (Grant No. [Number 3]).
```

**中文**：
```
經費來源：本研究承以下計畫補助：(1) [資助機構 1]（計畫編號：[編號 1]）；
(2) [資助機構 2]（計畫編號：[編號 2]）；(3) [資助機構 3]（計畫編號：[編號 3]），
特此致謝。
```

### 無資助

**英文**：
```
Funding: This research received no specific grant from any funding agency in
the public, commercial, or not-for-profit sectors.
```

**中文**：
```
經費來源：本研究未接受任何公部門、商業或非營利機構之特定補助。
```

### 部分資助

**英文**：
```
Funding: This work was partially supported by the [Funder] (Grant No. [Number]).
The funder had no role in study design, data collection and analysis, decision
to publish, or preparation of the manuscript.
```

**中文**：
```
經費來源：本研究部分經費由[資助機構]（計畫編號：[編號]）補助。
資助機構對本研究之設計、資料蒐集與分析、投稿決定及稿件撰寫均未介入。
```

## 與 COI 聲明的區別

Funding statement 與 Conflict of Interest（COI）statement 是**兩個獨立的聲明**，不可混為一談：

| 比較項目 | Funding Statement | COI Statement |
|---------|------------------|---------------|
| **目的** | 揭露研究資金來源 | 揭露可能影響研究客觀性的利益衝突 |
| **內容** | 資助機構名稱、計畫編號、資助角色 | 財務利益、諮詢關係、股權、專利等 |
| **無相關時** | 仍需聲明「無資助」 | 仍需聲明「無利益衝突」 |
| **期刊位置** | 通常在致謝之前或之後 | 通常在致謝之前或之後 |
| **揭露範圍** | 僅涵蓋本研究的直接資金 | 涵蓋所有可能影響客觀性的關係 |

### COI 聲明範本

**無利益衝突**（英文）：
```
Declaration of Interest: The authors declare that they have no known competing
financial interests or personal relationships that could have appeared to
influence the work reported in this paper.
```

**無利益衝突**（中文）：
```
利益衝突聲明：所有作者聲明，就本文所報導之研究而言，不存在任何已知的
競爭性財務利益或個人關係。
```

**有利益衝突**（英文）：
```
Declaration of Interest: [Author Name] has received research grants from
[Organization] and serves as a consultant for [Company]. The remaining
authors declare no competing interests.
```

## 期刊常見要求格式

### 依出版社

| 出版社 | 位置 | 格式要求 |
|--------|------|---------|
| **Elsevier** | 獨立 "Funding" 節 | Funder name + grant number；使用投稿系統的 funding 表格 |
| **Springer Nature** | "Declarations" 節下 | 含 Funding + COI + Ethics 等子節 |
| **Wiley** | "Acknowledgments" 或獨立節 | 依期刊而異 |
| **Taylor & Francis** | "Funding" 或 "Disclosure statement" | Funder name + grant number |
| **PLOS** | 投稿系統 + 稿件 | 投稿系統中逐筆填入；稿件中亦需包含 |
| **MDPI** | "Funding" 獨立節 | 位於 "Author Contributions" 之後 |

### 依主要高教期刊

| 期刊 | 格式 |
|------|------|
| *Higher Education* | Springer "Declarations" 格式 |
| *Studies in Higher Education* | Taylor & Francis "Disclosure statement" |
| *Research in Higher Education* | Springer "Declarations" 格式 |
| *Quality in Higher Education* | Taylor & Francis "Disclosure statement" |
| 教育科學研究期刊 | 中文格式，致謝段落中說明 |
| 高等教育 | 中文格式，致謝段落中說明 |

## 台灣特殊要求

### NSTC 計畫成果報告要求

1. **期刊論文致謝**：獲 NSTC 補助的研究成果發表時，**必須**在論文中致謝並列出計畫編號
2. **編號格式演變**：
   - 2022 年 7 月前：`MOST [年度]-[學門]-[類別]-[序號]`
   - 2022 年 7 月後：`NSTC [年度]-[學門]-[類別]-[序號]`
   - 引用時使用計畫核定時的機構名稱
3. **中英文均需聲明**：若論文為英文，funding statement 用英文；若為中文，用中文
4. **Open Access 費用**：若 APC 由計畫經費支付，部分期刊要求額外說明

### 教育部計畫要求

1. 高教深耕計畫成果需標註「教育部高等教育深耕計畫」
2. 教學實踐研究計畫成果需標註計畫編號
3. 部分計畫要求使用特定標示格式（依年度公告）

### 校內計畫

各校規定不同，建議：
1. 查詢研究發展處的致謝格式規範
2. 使用校名全稱（含英文全稱）
3. 標註校內計畫編號

## 最佳實務

### 撰寫 Funding Statement 的檢核清單

- [ ] 所有資助來源均已列出（含校內計畫）
- [ ] 計畫編號正確且完整
- [ ] 資助機構使用官方全稱（非縮寫首次出現時）
- [ ] 若無資助，明確聲明「無資助」
- [ ] 若資助機構要求特定 disclaimer，已逐字引用
- [ ] Funding statement 與 COI statement 分開陳述
- [ ] 英文論文使用英文聲明；中文論文使用中文聲明
- [ ] 多重資助來源已依主要性或字母排序
- [ ] 資助機構名稱反映計畫核定時的正式名稱
